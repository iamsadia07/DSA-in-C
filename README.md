# 📚 Data Structures in C

This repository contains basic implementations of **Data Structures in C**.
It is created for learning and practicing core DSA concepts step by step.

---

## 🔹 Topics Covered

### 🔗 Linked List

* Traversal
* Insertion (Beginning, End, After Node)
* Deletion (Beginning, End)
* Searching
* Counting nodes
* Reverse Linked List
* Loop Detection

---

### 📦 Stack

* Push
* Pop
* Display
* isEmpty
* isFull

---

### 🔁 Queue

* Enqueue
* Dequeue
* Display
* isEmpty

---

## 💡 Concepts Used

* Pointers
* Dynamic Memory Allocation (`malloc`)
* Structures in C
* Basic Operations of Data Structures

---

## 📁 Project Structure

* `LinkedList/`
* `Stack/`
* `Queue/`

Each folder contains separate programs for different operations.

---



## 🎯 Learning Goal

This project helps in understanding basic data structures and improves problem-solving skills in C.
