# 📚 Queue Implementation in C

This repository contains basic implementations of **Queue** using C programming language.

---

## 🔹 What is a Queue?

A Queue is a linear data structure that follows the **FIFO (First In First Out)** principle.

👉 The element inserted first is removed first.

---

## 🔧 Operations Implemented

* ✅ Enqueue (Insert element into queue)
* ✅ Dequeue (Remove element from queue)
* ✅ Display (Print queue elements)

---

## 💡 Types of Queue Implementation

* 📦 Queue using Array
* 🔗 Queue using Linked List

---

## 🧠 Concepts Covered

* FIFO principle
* Front and Rear pointers
* Dynamic memory allocation (`malloc`)
* Overflow and Underflow conditions
* Traversal in linked list

---


## 🎯 Learning Goal

This project helps in understanding queue operations and builds a strong foundation for further important topics.
