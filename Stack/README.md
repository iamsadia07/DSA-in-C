# 📚 Stack Implementation in C

This repository contains basic implementations of **Stack** using C programming language.

---

## 🔹 What is a Stack?

A Stack is a linear data structure that follows the **LIFO (Last In First Out)** principle.

👉 The last element inserted is the first one to be removed.

---

## 🔧 Operations Implemented

* ✅ Push (Insert element into stack)
* ✅ Pop (Remove element from stack)
* ✅ isEmpty (Check if stack is empty)
* ✅ isFull (Check if stack is full - array implementation)
* ✅ Display (Print stack elements)

---

## 💡 Types of Stack Implementation

* 📦 Stack using Array
* 🔗 Stack using Linked List

---

## 🧠 Concepts Covered

* LIFO principle
* Array-based implementation
* Pointer-based (Linked List) implementation
* Overflow and Underflow conditions
* Memory management using `malloc`

---


* **Overflow** → When stack is full and push is attempted
* **Underflow** → When stack is empty and pop is attempted

---

## 🎯 Learning Goal

This project helps in understanding stack operations and builds a strong foundation for advanced topics like recursion, expression evaluation, etc.
