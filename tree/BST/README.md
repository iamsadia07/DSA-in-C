# 🌳 Binary Search Tree (BST) in C

This repository contains the implementation of **Binary Search Tree (BST)**.
It helps in understanding how data is stored, searched, and managed efficiently using trees.

---

## 🔹 What is a BST?

A Binary Search Tree is a type of Binary Tree where:

👉 All nodes in the left subtree are smaller than the root
👉 All nodes in the right subtree are greater than the root

---

## 🌳 Example BST

```
      10
     /  \
    5    20
         /
        15
```

---

## 🔧 Operations Implemented

* ✅ Insertion
* ✅ Searching
* ✅ Inorder Traversal
* ✅ Preorder Traversal
* ✅ Postorder Traversal
* ✅ Find Minimum
* ✅ Find Maximum
* ✅ Height of BST

---

## 🧠 Key Concept

👉 Inorder traversal of BST always gives **sorted output**
