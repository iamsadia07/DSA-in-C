# 📚 Linked List in C

This repository contains basic implementations of **Linked List** in C programming language. It is helpful for beginners who are learning **Data Structures (DSA)** step by step.

---

## 🔹 What is a Linked List?

A Linked List is a linear data structure where elements (nodes) are connected using pointers.
Each node contains:

* **Data** (value)
* **Pointer** to the next node

---

## 🔧 Operations Implemented

* ✅ Traversal (Display elements)
* ✅ Insertion at Beginning
* ✅ Insertion at End
* ✅ Insertion after a given node
* ✅ Deletion from Beginning
* ✅ Deletion from End
* ✅ Search element in Linked List
* ✅ Count number of nodes
* ✅ Reverse Linked List
* ✅ Detect Loop (O) in Linked List

---



## 🎯 Learning Goal

This project helps in understanding how linked lists work internally and builds a strong base for advanced data structures.
